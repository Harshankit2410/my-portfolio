## ğŸŒ Deployment


### Deploying to Vercel

1. [Sign up](https://vercel.com/signup) or log in to [Vercel](https://vercel.com/).
2. Click **New Project** and import your GitHub/GitLab/Bitbucket repository.
3. Vercel will auto-detect the project as a Vite React app.
4. Set the following build settings if not auto-detected:
	- **Build Command:** `npm run build`
	- **Output Directory:** `dist`
5. Click **Deploy**.

Your site will be live on a Vercel URL after deployment. For custom domains, follow Vercel's domain setup instructions.
